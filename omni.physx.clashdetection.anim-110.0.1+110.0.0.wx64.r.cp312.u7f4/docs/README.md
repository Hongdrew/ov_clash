# Clash Detection Anim

Adds support for curve animations. Converts curve animations into discrete USD time samples, which is format that clash detection engine requires.

# Curve Animations
Curve animations are a method of representing changes in an object's properties over time using mathematical curves. These curves are defined by keyframes, which are specific points that determine the value of a property at a particular time.

# Time Samples
Time samples are used to represent time-varying data. They consist of a series of values (translation, rotation, scale) associated with specific frames, allowing for the encoding of animations as discrete samples over time.

For full documentation, please see the omni.physx.clashdetection.bundle extension.
