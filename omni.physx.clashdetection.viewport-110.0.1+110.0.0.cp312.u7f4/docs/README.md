# Clash Detection Viewport

Adds visualization for clashes in main viewport and a dedicated viewport to inspect results.

Show or hide the clash detection window:

Window -> Physics -> Clash Detection Viewport 

OR 

Window -> Simulation -> Clash Detection Viewport.
To view it in action select any clash in the `Clash Detection` table user interface.

For full documentation, please see the omni.physx.clashdetection.bundle extension.
